  Your text files saved localy and encrypted. Keep your passwords, text documents and

Secret Scribble is an essential tool that mainly allows you to create ecrypted files in text format, in any language and using the powerfull AES encryption algorithm. You can access your files only by loggin in, offline. It has a built in text editor and a password generator. It can decript and export your files, or let's you view and edit them directly. On top of that, the app provides some other useful features that make it act as a basic file manager. Like disk info, file browsing / searching / execution to maximize efficiency.
Please note that the app is in early development. It will keep improving update by update, taking heavy inspiration from user feedback.

Why choose Secret Scribble over anything else:




1. Secret Scribble, just like most password managers, offers a range of standard features such as encryption, a password generator, secure password sharing, two-factor authentication (2FA), password health checks, import/export functionality, and a user-friendly interface.

2. Secret Scribble offers enhanced reliability and security, setting itself apart with a strong emphasis on offline access. This approach significantly improves security by storing your information locally on your device, rather than with other reputable online password managers that are often targets for cyberattacks. Additionally, Secret Scribble includes features such as automatic internet disconnection and an on-screen keyboard, providing protection against keylogging and unauthorized screen recording or video capture while you enter your credentials.

3. Managing passwords offline is often more sophisticated. Using tools like OpenSSL, terminals, and commands to encrypt and decrypt text can be tedious, and any software that automates this process tends to lack user-friendly design and functionality. Secret Scribble provides a more minimal environment and user friendly experience.

4. Secret Scribble is free! It will never force you to pay in order to unlock, access, or modify your data.
