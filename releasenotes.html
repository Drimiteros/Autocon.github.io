MyWidget Version Alpha 2.1

1/4/24



Features:

1. New rendering method allows the widgets have smoother edges.
2. Widgets support multiple levels of transparency.
3. The user can now edit the properties of each widget (position, scale, color, etc.) directly and in real time from the save file located at "src/Saves/...". Every widget can now be disabled by changing the "Is active:1" to "Is active:0" located in the save file of each widget.




Bug Fixes:

1. Fixed the black screen bug.
2. Fixed widget flickering.
3. Fixed a bug that made the widgets to overlap with other windows.
4. Fixed choppy framerate. Overlaping windows no longer appear moving choppy when the user drags them across the screen.




MyWidget Version Alpha 2.0_1

28/3/24



Bug Fixes:

1. Fixed a bug that allowed the user to accidentally interact with the current widget color while not on edit mode.



Bugs:

1. Black screen: When you set the app to automaticaly start with windows there is a possibility for the screen to turn black with only the widgets visible, also this happens if you click on top of the widget. To fix just click somewhere in the screen.

2. Choppy frames: In order to calculate the CPU utilization, the program pauses for some milliseconds, this makes other overlaping windows to appear choppy when moving them. A fix is on its way.

MyWidget Version Alpha 1.1

15/3/24



Features:

1. Save preferences: Now your widget preferences like scale, color and position are stored in the app's folder under "src/Saves/" and applied automaticaly when the app starts. You can edit the save files directly if you want to and add custom colors or change the scale of the widgets and more.






1. "Load: 0" resets the rest of the values underneath to their defaults and "Load: 1" keeps them intact upon the start of the app.
2. PositionX/Y save the widget's position in the screen.
3. R,G,B store the color values of the widget.
4. Character Size stores the size of the widget's letters.
